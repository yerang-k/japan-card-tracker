# 일본 카드 트래커

일본 여행 중 **결제·ATM 인출을 할 때 어떤 카드가 가장 유리한지** 금액을 넣으면 바로 비교해 주는 웹앱입니다. 쓴 내역과 이번 달 혜택 한도도 함께 관리합니다.

> 🔗 **바로 쓰기** — <https://winter-term-7a22.yeahskyr.workers.dev/>

설치가 필요 없고, 기록은 **내 폰(브라우저)에만 저장**됩니다. 서버로 전송되지 않습니다.

---

## 무엇을 해 주나

- **금액을 넣으면 카드별 예상 혜택을 비교**하고 가장 유리한 카드를 추천
- **결제 / ATM 인출**을 나눠서 계산 (인출 가능한 카드만 활성화)
- **혜택 한도 현황**을 게이지로 표시 — 한도를 넘으면 자동으로 다음으로 유리한 카드를 추천
- **현금 지갑** — 인출·현금결제를 반영한 잔액과 원화 환산
- **여행 요약** — 총 지출과 누적 혜택
- **실시간 환율 자동 반영** (12시간마다 갱신, 오프라인이면 마지막 값 사용)

## 등록된 카드와 계산 기준

| 카드 | 결제 | ATM 인출 |
|---|---|---|
| **NH zgm.일본여행중** (JCB) | 일본 오프라인 **10% 적립** · 해외서비스수수료 면제 | 해당 없음 |
| **네이버페이** 머니카드 | **2% 적립** (수수료는 포인트 환급) | 해당 없음 |
| **네이버페이 QR** (Alipay+) | 적립 없음 · 수수료 0 · 좋은 환율 | 해당 없음 |
| **토스뱅크** | 2% 캐시백 − 수수료(1% + $0.5) | 월 5회·$700까지 무료, 초과 시 1% + $3 |
| **트래블로그** | 수수료 0 · 적립 없음 | 세븐뱅크 무료 · 횟수 무제한 |
| **트래블월렛** | 수수료 0 · 적립 없음 | 월 $500까지 무료, 초과분 2% |
| **현금** | 현금 지갑에서 차감 | — |

### NH 카드 이용 조건 (중요)

- **전월실적 30만원 이상**이어야 10% 적립이 적용됩니다
- **연간 10만 포인트 한도** (결제액 약 100만원 = 약 ¥109,000). 앱이 남은 한도를 추적해 소진되면 다른 카드를 추천합니다
- **일본 오프라인 매장**만 10%이며, 온라인 결제는 별도 기준입니다

## 알아두세요

- 카드사 정책은 수시로 바뀝니다. 계산 기준은 **2026년 7월 시점**이며, 실제 청구액과 다를 수 있습니다.
- 네이버페이 QR은 Alipay+ 가맹점에서만 됩니다. 받지 않는 가게에서는 다음으로 유리한 카드를 쓰세요.
- ATM 기기수수료(¥110~220)는 계산에 포함되어 있지 않습니다. 트래블월렛은 **이온뱅크 ATM**에서만 기기수수료가 면제됩니다.

의존성 없는 단일 HTML 파일입니다. 바닐라 JavaScript, `localStorage`, [open.er-api.com](https://open.er-api.com) 무료 환율 API(키 불필요).
